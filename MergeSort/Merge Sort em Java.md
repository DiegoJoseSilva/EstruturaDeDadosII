# **Merge Sort em Java**

## **📌 Descrição**

Este projeto apresenta a implementação do algoritmo de ordenação **Merge Sort** na linguagem Java, utilizando o paradigma **Dividir para Conquistar**.

O programa gera um vetor de números inteiros aleatórios, exibe o vetor original, realiza a ordenação utilizando o Merge Sort e, por fim, exibe o vetor ordenado.

---

## **⚙️ Tecnologias utilizadas**

* Java  
* IntelliJ IDEA

---

## **🚀 Como executar**

1. Abra o projeto no IntelliJ IDEA  
2. Localize o arquivo `MergeSort.java`  
3. Execute a classe (botão ▶️)

---

## **🧪 Funcionamento**

O programa realiza os seguintes passos:

1. Gera um vetor com números aleatórios  
2. Exibe o vetor original  
3. Aplica o algoritmo Merge Sort  
4. Exibe o vetor ordenado

---

## **🧠 Sobre o algoritmo**

O **Merge Sort** é um algoritmo de ordenação eficiente baseado na estratégia de **Dividir para Conquistar**:

* Divide o vetor em partes menores  
* Ordena cada parte recursivamente  
* Intercala (merge) as partes ordenadas

### **📊 Complexidade**

* Tempo: O(n log n)  
* Espaço: O(n)

---

## **📷 Exemplo de saída**

Vetor original:  
\[45, 12, 78, 3, 56, 23, 9, 1, 67, 34\]

Vetor ordenado:  
\[1, 3, 9, 12, 23, 34, 45, 56, 67, 78\]
